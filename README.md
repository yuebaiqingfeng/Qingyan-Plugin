清言群管插件
智能群聊禁言管控插件，提供智能自动禁言和手动指令禁言功能，所有回复均由模型自主生成，无固定模板。
功能特性
智能 LLM 判定：根据聊天内容智能判断是否需要禁言，自动处理违规用户
灵活时长管理：支持自定义最短 / 最长禁言时长，自动修正超出范围的时长
自然回复：所有场景回复均由模型自主生成，没有固定模板，完全贴合人设
完整权限控制：
超级管理员：配置的超级管理员无法被禁言
授权管理员：仅授权管理员可以执行禁言操作
群组权限：可以限制仅指定群组可以使用该功能
手动指令：支持手动禁言指令，方便管理员快速操作
参数验证：完整的输入参数校验，避免错误操作
指令说明
手动禁言指令
你可以使用 /qy 指令手动执行禁言操作，用法：
/qy <用户名> <时长(秒)> [理由]

使用示例
/qy 用户名 300：禁言指定用户 300 秒
/qy 张三 600 刷屏：禁言张三 600 秒，理由为刷屏
/qy @某人 1800 违规内容：禁言 @的用户 1800 秒，理由为违规内容
配置说明
组件配置
enable_qingyan_action：是否启用智能自动禁言功能（默认开启）
enable_qingyan_command：是否启用手动禁言指令（默认关闭）
权限配置
admin_users：无法被禁言的超级管理员列表，格式为 ['plat:uid']，例如 ['qq:123456789']
allowed_users：有权限执行禁言操作的授权管理员列表，格式为 ['plat:uid']，例如 ['qq:123456789']
allowed_groups：允许使用禁言功能的群组列表，格式为 ['plat:gid']，例如 ['qq:123456789']
核心功能配置
min_duration：最短禁言时长（秒），默认 60 秒，低于该值的时长会自动修正
max_duration：最长禁言时长（秒），默认 2592000 秒（30 天），高于该值的时长会自动修正
